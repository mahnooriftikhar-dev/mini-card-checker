# 💳 Mini Card Checker

A simple **Mini Card Checker** project built using **HTML, CSS, and JavaScript**.
This project checks and validates card details entered by the user using basic JavaScript logic..

---

## 🚀 Features

* Validate card input fields
* Check card number format
* Verify expiry date and CVV input
* Display messages based on valid or invalid input
* Simple and clean user interface

---

## 🛠️ Technologies Used

* **HTML** — Structure of the card form
* **CSS** — Styling and layout design
* **JavaScript** — Validation logic and functionality

---

## .gitattributes File

Create a file named **`.gitattributes`** in your project folder and add:

```
*.css linguist-detectable=false
```

This helps GitHub focus more on **JavaScript** in language statistics.
